## Pair Skills with Scheduled Tasks

Skills become even more powerful when you combine them with Cowork's scheduled tasks. A scheduled task runs automatically on a schedule you set, so Claude does work for you in the background without you being there.

Some examples:

- **Morning Briefing + daily schedule** — Set it to run every morning at 7am. Wake up to a full briefing of what's happening in your niche, already sitting in your folder.
- **Receipt Scanner + weekly schedule** — Drop receipts into a folder all week, then have Claude scan and build your expense report every Friday.
- **Quick Research + on demand** — Save a research task you can fire off from your phone with Dispatch whenever a competitor drops something or news breaks.
- **Budget Planner + monthly schedule** — Auto-generate an updated budget report on the 1st of every month.

To set one up, just tell Claude: "Schedule the morning briefing to run every day at 7am" and it handles the rest. You can also trigger any scheduled task manually whenever you want.

The combination of skills (what Claude knows how to do) and scheduled tasks (when Claude does it) is what turns Cowork from a chatbot into an actual system that works for you.
